## ğŸš€ Installation & Build

Since this extension is in a developer phase, follow these instructions to install it:

1. Clone or download this repository.
2. Ensure you have [Node.js](https://nodejs.org/) installed.
3. Run `npm install` to download build dependencies (Vite & TypeScript).
4. Run `npm run build` to compile the TypeScript and assets into the `/dist` directory.
5. In your browser, install the unpacked extension loading the `dist/` directory. For a detailed guide on how to do this in different web browsers, refer to [Installation.md](./Installation.md).

## ğŸ’¡ Usage

Once installed, simply open a **New Tab** or click the extension's icon in the browser toolbar to access your **Group Speed Dial Pro** interface. From there, you can:
- **Right-click** outside dials or inside groups to instantly re-arrange sorting logic.
- Click the top right **Global Settings Menu** for Customization, Log reading, Statistics, and Help files.
- Press **Settings** within a Group Tab to change its core aesthetics and security constraints independently.
